# Sales Performance Analysis

## Overview

This project analyzes retail sales data to identify revenue trends, profit, customer behaviour and business insights using SQL and Power BI.

## Business Problem

Businesses generate thousands of transactions every month. This project helps identify:

- Top selling products
- Best performing regions
- Monthly sales trend
- Profit analysis
- Customer insights

## Tools

- SQL
- Excel
- Power BI

## KPIs

- Total Sales
- Total Profit
- Total Orders
- Average Order Value

## Dashboard

- Sales Trend
- Category Analysis
- Region Analysis
- Customer Analysis

## Business Recommendations

- Increase investment in profitable products.
- Improve sales in low-performing regions.
- Reduce discounts on low-margin products.
- Improve customer retention.
